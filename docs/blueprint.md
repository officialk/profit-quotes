# **App Name**: ProfitPro Calculator

## Core Features:

- Profit Calculation from Quoted Price: Calculate the profit based on a quoted price and fixed or percentage-based expenses.
- Quoted Price Calculation from Target Profit: Determine the required quoted price to achieve a target profit, considering fixed or percentage-based expenses.
- Expense Input Interpretation: Intelligently interpret expenses entered as either a fixed numeric amount or a percentage of the quoted price using an AI tool to distinguish between numbers and expressions such as percentages.
- Automatic Rounding: Automatically round all calculated outputs to two decimal places for clarity and precision.
- Calculation Mode Selection: Provide a clear interface to select between 'Profit From Quoted Price' and 'Quoted Price From Target Profit' calculation modes.
- Reset Button: Include a 'Reset' button to clear all fields and start a new calculation.
- Descriptive Label: Allow the user to save an arbitrary string with each estimate so the results can easily be looked up.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) for trustworthiness and financial focus.
- Background color: Light gray (#E9EAED) for a clean, professional feel.
- Accent color: Soft purple (#8E44AD) for highlighting key values and call-to-action buttons.
- Body and headline font: 'Inter' sans-serif font for clear readability.
- Use simple, clean icons related to finance and calculations.
- Employ a clear, intuitive layout that separates input fields and results, ensuring ease of use.
- Use subtle animations to provide feedback on calculations and enhance the user experience.
